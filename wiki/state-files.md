# state files

- `IDENTITY.md` — mission and constitution
- `PERSONALITY.md` — voice and style constraints
- `JOURNAL.md` — append-only session memory
- `LEARNINGS.md` — external learnings memory
- `DAY_COUNT` — numeric evolution counter
- `SESSION_PLAN.md` — ephemeral planning artifact
- `ISSUES_TODAY.md` — ephemeral issue digest
- `ISSUE_RESPONSE.md` — ephemeral response payload
